Which body parts have reached MMI? (i)

To proceed with this PR-4 report, please select the body parts that have reached MMI. Then, select which body parts you want to have rated in this PR-4 report.

Body parts that have not reached MMI will not be included in this report and will not receive an impairment rating.

You cannot continue with this report unless "Do you want an impairment rating today for this body part?" is set to "Yes" for one or more body parts.
